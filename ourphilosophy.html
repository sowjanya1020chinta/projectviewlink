Design

Software design is the process of dessigning the elements of a software such as the architecture, modules and components, different intefaces of these comoents and the data that goes into it.

Development

Software development refers to a set of computer science activities dedicated to the process of creating, dessigning, deploying and supporting software. Software itself is the set of instruction or programs that tell a computer what to do.

Deployment

Software deployment is the process of making software avaliable to be used on a system by users and other programs. You might deploy software to create a backup copy of the software to another system, or to create another SMP/E-servericeable copy for installing service or other products.

Application supoort

Application support usually offeers technical support and human support. Technical support relates to the maintenance of the app, software and other organization's technological systems. Human support relates to helping the end-user understand how to use the application or system and help solve any issue.
